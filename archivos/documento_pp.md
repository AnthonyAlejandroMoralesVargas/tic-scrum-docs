# Product Planning - PoliTutorías
### 1. Visión del producto

### 1.1 Áreas de valor

| Área               | Subcategoría                                         | Justificación Interna (Valor)                                                                                                                                                                                       |
| ------------------ | ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Habilitador        | Target a new market                                  | Abre un mercado de tutorías exclusivo para estudiantes de la EPN, al establecer un canal formal que conecta a estudiantes que necesitan tutorías con otros capaces de ser tutores.                                  |
| Diferenciador      | Differentiate from competitors                       | A diferencia de las alternativas informales, se introduce un sistema de reputación y perfiles detallados de tutores para resolver la preocupación de los estudiantes sobre la calidad y fiabilidad de los tutores . |
| Reductor de costos | Reduce the number of people or their time allocation | Optimiza el tiempo de estudiantes y tutores al centralizar la búsqueda, gestión de agenda y coordinación. Elimina el tiempo perdido y la falta de visibilidad en canales informales.                                |

### 1.2 Visión con las áreas de valor

- **Para:** estudiantes de la Escuela Politécnica Nacional,
- **Que:** buscan recibir o impartir tutorías,
- **El:** producto PoliTutorías
- **Es una:** aplicación web para la gestión de tutorías,
- **Que:** optimiza el tiempo de tutores y estudiantes centralizando la publicación y solicitud de tutorías.
- **A diferencia de:** Learning Center USFQ
- **Nuestro producto:** permite construir una reputación para los tutores a través de reseñas que ayudan a los estudiantes a tomar decisiones.

### 2. Product Backlog de alto nivel

| Nro. | Título                          | Historia de Usuario                                                                                                                                     |
| ---- | ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| EP01 | Publicar oferta tutorías       | Como tutor,quiero comunicar mi domino de una materia para atraer estudiantes interesados.                                                               |
| EP02 | Revisar oferta tutorías         | Como estudiante, quiero revisar la oferta de tutorías junto con la información del tutor para verificar la credibilidad del tutor antes de contactarlo. |
| EP03 | Gestión de agenda               | Como tutor, quiero gestionar la disponibilidad de mi agenda para que los estudiantes puedan reservar tutorías en horarios que me resulten convenientes. |
| EP04 | Agendar tutorías                | Como estudiante, quiero agendar una tutoría para recibir ayuda de un tutor en una sesión personal.                                                      |
| EP05 | Gestión solicitudes             | Como tutor, quiero gestionar las solicitudes de tutoría para brindar una respuesta a los estudiantes que solicitan mis servicios.                       |
| EP06 | Reputación tutor                | Como estudiante, quiero dejar una reseña a un tutor para compartir mi experiencia sobre la tutoría.                                                     |
| EP07 | Gestión cobro                   | Como tutor, quiero administrar el cobro de mis tutorías, para recibir el pago de forma segura.                                                          |
| EP08 | Gestión de reputación del tutor | Como tutor, quiero solicitar la mediación de reseñas injustas, para proteger mi reputación de comentarios malintencionados.                             |
| EP09 | Reputación estudiante           | Como tutor, quiero registrar penalizaciones a estudiantes para alertar tutores sobre el comportamiento del estudiante.                                  |
| EP10 | Material apoyo del tutor        | Como tutor, quiero proporcionar material de apoyo a los estudiantes para que puedan repasar los temas por su cuenta.                                    |
| EP11 | Chat conversacional             | Como estudiante, quiero conversar con el tutor para agilizar la coordinación de la tutoría.                                                             |
